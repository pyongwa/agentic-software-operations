# Executive variant

A shorter, decision-oriented cut of the same report for an executive audience.

## Structure
- **One-line headline** — the single most important state change this period.
- **On track / at risk / blocked** — 1–2 bullets each, only what an exec must know.
- **Decisions needed** — explicit asks, each with an owner and a by-when.

## Rules
- Derived from the same verified data as the full report — never a separate set of
  numbers.
- Omit detail, never change the facts. Keep verified-vs-inferred honesty.
- Draft-first; same approval gate as the full report.
